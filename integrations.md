# Integrations

Defacta is built to drop into existing AI workflows in whatever form fits.

## REST API

Call Defacta from any backend, batch job, or serverless function. Send text, receive a structured JSON verdict. This is the most flexible way to embed verification into your own product.

## MCP server

Defacta speaks the Model Context Protocol, so AI agents and MCP-aware tools can call it as a native tool. Give an agent the ability to fact-check its own output before it acts on it.

## Browser extension

Verify content in place while you browse. The extension runs Defacta against what you are reading and surfaces its findings inline - useful for research, review, and spotting AI-written misinformation.

> These integration surfaces are described from the Defacta website for this demo preview. Setup steps and auth details live in the official documentation.
